# Turnstile Gate Assets

Place your custom turnstile gate PNG assets in this folder using the exact filenames:

- turnstileGate-r0.png
- turnstileGate-r1.png
- turnstileGate-r2.png
- turnstileGate-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the gate centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
