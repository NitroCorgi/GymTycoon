# Turnstile Assets

Place your custom turnstile PNG assets in this folder using the exact filenames:

- turnstile-r0.png
- turnstile-r1.png
- turnstile-r2.png
- turnstile-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the turnstile centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
