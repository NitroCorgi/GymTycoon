# Queue Stanchion Assets

Place your custom queue stanchion PNG assets in this folder using the exact filenames:

- queueStanchion-r0.png
- queueStanchion-r1.png
- queueStanchion-r2.png
- queueStanchion-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the stanchion centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
