# Front Desk Assets

Place your custom front desk PNG assets in this folder using the exact filenames:

- frontDesk-r0.png
- frontDesk-r1.png
- frontDesk-r2.png
- frontDesk-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the desk centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
