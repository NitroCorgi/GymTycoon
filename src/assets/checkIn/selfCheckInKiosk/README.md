# Self Check-In Kiosk Assets

Place your custom self check-in kiosk PNG assets in this folder using the exact filenames:

- selfCheckInKiosk-r0.png
- selfCheckInKiosk-r1.png
- selfCheckInKiosk-r2.png
- selfCheckInKiosk-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the kiosk centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
