# Reception Desk Assets

Place your custom reception desk PNG assets in this folder using the exact filenames:

- receptionDesk-r0.png
- receptionDesk-r1.png
- receptionDesk-r2.png
- receptionDesk-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the desk centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
