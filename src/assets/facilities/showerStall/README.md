# Shower Stall Assets

Place your custom shower stall PNG assets in this folder using the exact filenames:

- showerStall-r0.png
- showerStall-r1.png
- showerStall-r2.png
- showerStall-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the stall centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
