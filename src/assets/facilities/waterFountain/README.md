# Water Fountain Assets

Place your custom water fountain PNG assets in this folder using the exact filenames:

- waterFountain-r0.png
- waterFountain-r1.png
- waterFountain-r2.png
- waterFountain-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the fountain centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
