# Locker Row Assets

Place your custom locker row PNG assets in this folder using the exact filenames:

- lockerRow-r0.png
- lockerRow-r1.png
- lockerRow-r2.png
- lockerRow-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the locker row centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
