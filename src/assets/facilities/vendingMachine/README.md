# Vending Machine Assets

Place your custom vending machine PNG assets in this folder using the exact filenames:

- vendingMachine-r0.png
- vendingMachine-r1.png
- vendingMachine-r2.png
- vendingMachine-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the machine centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
