# Shower Assets

Place your custom shower PNG assets in this folder using the exact filenames:

- shower-r0.png
- shower-r1.png
- shower-r2.png
- shower-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the shower centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
