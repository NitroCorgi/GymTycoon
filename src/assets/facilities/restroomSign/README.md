# Restroom Sign Assets

Place your custom restroom sign PNG assets in this folder using the exact filenames:

- restroomSign-r0.png
- restroomSign-r1.png
- restroomSign-r2.png
- restroomSign-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the sign centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
