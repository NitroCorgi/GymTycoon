# Towel Station Assets

Place your custom towel station PNG assets in this folder using the exact filenames:

- towelStation-r0.png
- towelStation-r1.png
- towelStation-r2.png
- towelStation-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the station centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
