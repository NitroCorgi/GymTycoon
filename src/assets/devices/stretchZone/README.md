# Stretch Zone Assets

Place your custom stretch zone PNG assets in this folder using the exact filenames:

- stretchZone-r0.png
- stretchZone-r1.png
- stretchZone-r2.png
- stretchZone-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
