# Bench Press Assets

Place your custom bench press PNG assets in this folder using the exact filenames:

- benchPress-r0.png
- benchPress-r1.png
- benchPress-r2.png
- benchPress-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
