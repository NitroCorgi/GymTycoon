# Rowing Machine Assets

Place your custom rowing machine PNG assets in this folder using the exact filenames:

- rowingMachine-r0.png
- rowingMachine-r1.png
- rowingMachine-r2.png
- rowingMachine-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
