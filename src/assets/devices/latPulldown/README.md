# Lat Pulldown Assets

Place your custom lat pulldown PNG assets in this folder using the exact filenames:

- latPulldown-r0.png
- latPulldown-r1.png
- latPulldown-r2.png
- latPulldown-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
