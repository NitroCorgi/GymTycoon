# Massage Chair Assets

Place your custom massage chair PNG assets in this folder using the exact filenames:

- massageChair-r0.png
- massageChair-r1.png
- massageChair-r2.png
- massageChair-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
