# Chest Press Machine Assets

Place your custom chest press machine PNG assets in this folder using the exact filenames:

- chestPress-r0.png
- chestPress-r1.png
- chestPress-r2.png
- chestPress-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
