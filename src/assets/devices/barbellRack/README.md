# Barbell Rack Assets

Place your custom barbell rack PNG assets in this folder using the exact filenames:

- barbellRack-r0.png
- barbellRack-r1.png
- barbellRack-r2.png
- barbellRack-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
