# Battle Ropes Assets

Place your custom battle ropes PNG assets in this folder using the exact filenames:

- battleRopes-r0.png
- battleRopes-r1.png
- battleRopes-r2.png
- battleRopes-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
