# Power Rack Assets

Place your custom power rack PNG assets in this folder using the exact filenames:

- powerRack-r0.png
- powerRack-r1.png
- powerRack-r2.png
- powerRack-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
