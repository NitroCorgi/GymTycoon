# Kettlebell Set Assets

Place your custom kettlebell set PNG assets in this folder using the exact filenames:

- kettlebellSet-r0.png
- kettlebellSet-r1.png
- kettlebellSet-r2.png
- kettlebellSet-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
