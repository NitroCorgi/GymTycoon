# Dumbbell Station Assets

Place your custom dumbbell station PNG assets in this folder using the exact filenames:

- dumbbellStation-r0.png
- dumbbellStation-r1.png
- dumbbellStation-r2.png
- dumbbellStation-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
