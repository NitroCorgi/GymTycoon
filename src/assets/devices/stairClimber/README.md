# Stair Climber Assets

Place your custom stair climber PNG assets in this folder using the exact filenames:

- stairClimber-r0.png
- stairClimber-r1.png
- stairClimber-r2.png
- stairClimber-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
