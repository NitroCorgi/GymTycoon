# Smith Machine Assets

Place your custom smith machine PNG assets in this folder using the exact filenames:

- smithMachine-r0.png
- smithMachine-r1.png
- smithMachine-r2.png
- smithMachine-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the device centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
