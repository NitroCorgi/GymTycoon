# Treadmill Assets

Place your custom treadmill PNG assets in this folder using the exact filenames:

- treadmill-r0.png
- treadmill-r1.png
- treadmill-r2.png
- treadmill-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the treadmill centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
