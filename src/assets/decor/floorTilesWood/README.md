# Floor Tiles Wood Assets

Place your custom floor tiles wood PNG assets in this folder using the exact filenames:

- floorTilesWood-r0.png
- floorTilesWood-r1.png
- floorTilesWood-r2.png
- floorTilesWood-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the tiles centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
