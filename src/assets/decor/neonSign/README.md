# Neon Sign Assets

Place your custom neon sign PNG assets in this folder using the exact filenames:

- neonSign-r0.png
- neonSign-r1.png
- neonSign-r2.png
- neonSign-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the sign centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
