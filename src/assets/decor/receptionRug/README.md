# Reception Rug Assets

Place your custom reception rug PNG assets in this folder using the exact filenames:

- receptionRug-r0.png
- receptionRug-r1.png
- receptionRug-r2.png
- receptionRug-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the rug centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
