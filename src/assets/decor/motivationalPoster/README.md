# Motivational Poster Assets

Place your custom motivational poster PNG assets in this folder using the exact filenames:

- motivationalPoster-r0.png
- motivationalPoster-r1.png
- motivationalPoster-r2.png
- motivationalPoster-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the poster centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
