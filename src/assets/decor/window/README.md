# Wallpaper White Assets

Place your custom wallpaper white PNG assets in this folder using the exact filenames:

- wallpaperWhite-r0.png
- wallpaperWhite-r1.png
- wallpaperWhite-r2.png
- wallpaperWhite-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the wallpaper centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
