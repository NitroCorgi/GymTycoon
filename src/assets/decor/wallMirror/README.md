# Wall Mirror Assets

Place your custom wall mirror PNG assets in this folder using the exact filenames:

- wallMirror-r0.png
- wallMirror-r1.png
- wallMirror-r2.png
- wallMirror-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the mirror centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
