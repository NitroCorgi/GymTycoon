# Potted Plant Assets

Place your custom potted plant PNG assets in this folder using the exact filenames:

- pottedPlant-r0.png
- pottedPlant-r1.png
- pottedPlant-r2.png
- pottedPlant-r3.png

Recommended dimensions per file: **256x256 px**.

Notes:
- Use transparent background.
- Keep the plant centered in frame.
- Keep all 4 rotations at identical canvas size and visual scale.
